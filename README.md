# Projeto
API REST de um CRUD de produtos e categorias.  Desenvolvido em python utilizando os frameworks Django e DjangoREST

## Tecnologias
Tecnologias utilizadas no projeto:
- Linguagem de programação Python versão 3.6.7 [página oficial](https://www.python.org/)
- Framework Django versão 2.1.7 [página oficial](https://www.djangoproject.com/)
- Django REST framework versão 3.9.2 [página oficial](https://www.django-rest-framework.org/)
- Editor Visual Studio Code versão 1.32 [página oficial](https://code.visualstudio.com/)
- Docker Community Edition (CE) versão 18.09.3 [documentação](https://docs.docker.com/install/linux/docker-ce/ubuntu/)
- Docker Compose versão 1.23.2 [documentação](https://docs.docker.com/compose/django/)
- Guia de estilos python PEP 8 [documentação](https://www.python.org/dev/peps/pep-0008/)

## Como rodar

#### 1 - Clonar o projeto
```bash
git clone https://github.com/rikrdogil/Backend-test.git
```

#### 2 - Iniciar o container usando um dos comandos abaixo( na pasta teste_api_rest)
```bash
docker-compose up       #rodando em foreground
docker-compose up -d    #rodando em background
```
